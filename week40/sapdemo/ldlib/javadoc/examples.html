SAP Java Connector 3.1
In order to try out the examples copy the full folder as is to a place, where you can compile the sources and run the classes. Alternatively, create a new eclipse project that uses the examples folder as root folder.
 
Examples
Beginner client examples This package contains simple examples that demonstrate calls from a Java application to an SAP ABAP Application Server.
Advanced client examples This package contains examples that demonstrate calls from a Java application to an SAP ABAP Application Server using some advanced features.
Beginner server examples This package contains simple examples that demonstrate dispatching of calls originating from an SAP ABAP Application Server and how to implement a function module in Java.
Beginner server stateful example This package contains a simple example that demonstrate dispatching of a stateful call sequence originating from an SAP ABAP Application Server.
Advanced server examples This package and its subpackages contain examples that demonstrate dispatching of calls originating from an SAP ABAP Application Server using some advanced features.
Sample destination and server configurations Sample configuration files that can be used with the default implementations of DestinationDataProvider and ServerDataProvider that are used in the examples. The default implementations are thought for fast proof-of-concepts, tests, and other non-productive usages.
